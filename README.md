# INTERIM - INTERmedial Image-text Model
This repository hosts the documentation for the INTERIM ontology.
<hr>
INTERIM is a conceptual model and onology for describing intermedial relations within the cultural heritage domain, with a focus on the text-image relations. 
In this, it integrates **GEkO (General Ekphrastic Ontology)** specifically developed for describing ekphrasis, i.e. the verbal description of a visual artwork.
<br> 
The documentation for GEkO is available at the following link: https://github.com/MariaFrancesca6/geko/tree/main
